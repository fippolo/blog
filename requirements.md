# blog
in this folder there is a github repository in wich i write using obsidian, that is my diary where i write about stuff i make, i would like to make a blog out of it, wich updates autonomously using the git repository to pull new content from github as i write it

- it has to be contained in docker to be able to self host 
- it has to be in a separate repository from the diary itself
- it has to be able to be used with any repository as long as it is formated like the example
- the format should be described in the readme 
- it should not require github authentication to pull the diary if the repository is public 
- has to work both in https and http 
- has to be able to use certificates using certbot 
- complexity should be kept as low as possible
